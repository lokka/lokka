## Ruby Installation

<h3>Windowsの場合</h3>

<p><a href="http://rubyinstaller.org/">RubyInstaller for Windows</a>を使うのが便利です。</p>

<p><a href="http://www.flickr.com/photos/komagata/5012318696/" title="rubyinstaller for windows web site by komagata, on Flickr"><img src="http://farm5.static.flickr.com/4083/5012318696_44eb451b2f_o.jpg" alt="rubyinstaller for windows web site" width="463" height="362"></a></p>

<p><a href="http://rubyinstaller.org/downloads/">Downloadsページ</a>からRuby 1.8.7をダウンロードします。</p>

<p><img src="http://farm5.static.flickr.com/4149/5011716751_ea07859002_o.png" alt="rubyinstaller for windows 1" width="513" height="398"></p>

<p>インストーラーを実行します。</p>

<p><img src="http://farm5.static.flickr.com/4091/5012318850_0e386e972b_o.png" alt="rubyinstaller for windows 2" width="513" height="398"></p>

<p>利用規約に合意したら"I accept the license"を選択して次へ(Next)へ。</p>

<p><img src="http://farm5.static.flickr.com/4092/5011760167_40e4d00646_o.png" alt="RubyInstaller for windows 3" width="513" height="398"></p>

<p>Rubyのインストールフォルダを決定し(デフォルトで構いません)、"Add Ruby executables to your PATH"(RubyにPATHを通す)と"Associate .rb and .rbw files with this Ruby installation"(.rbファイル、.rbwファイルをRubyと関連付ける)にチェックを入れてインストール(Install)。</p>

<p><img src="http://farm5.static.flickr.com/4092/5012319004_76c65a83be_o.png" alt="rubyinstaller for windows 4" width="513" height="398"></p>

<p>これでRubyとRubyGems(周辺ライブラリ)がインストールされました。</p>

<h3>Mac OS Xの場合</h3>

<p>標準でインストールされています。</p>

<h3>Ubuntuの場合</h3>

<p>端末(ターミナル)から下記を実行。</p>

<pre><code>$ sudo aptitude install ruby rubygems</code></pre>

<p>もしくはUbuntuソフトウェアセンターからrubyとrubygemsをインストール。</p>
