# Lokka

A lightweight, flexible CMS written in Ruby.
Built with Sinatra and ActiveRecord, designed for simplicity and ease of deployment.

[View on GitHub](https://github.com/lokka/lokka){: .btn .btn-primary }
[Documentation](https://github.com/lokka/lokka/wiki){: .btn }

## Features

### Simple
Easy to install, configure, and deploy. Get your blog running in minutes.

### Flexible
Create themes with ERB or Haml. Extend with plugins using Sinatra extensions.

### Modern
Ruby 3.2+, ActiveRecord, and easy deployment with Kamal or Docker.
