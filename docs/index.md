## CMS for Cloud

Lokka runs not only on Windows, Mac OSX, and UNIX variants, but also in cloud environments like [Heroku](https://www.heroku.com/), [Sqale](https://sqale.jp/) and [Amazon EC2](https://aws.amazon.com/ec2/). You can operate a practical website for free.
