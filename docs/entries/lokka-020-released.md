## Lokka 0.2.0 Released

<p><a href="http://www.flickr.com/photos/komagata/5401134396/" title="Screen shot 2011-01-30 at 22.52.33 by komagata, on Flickr"><img src="http://farm6.static.flickr.com/5180/5401134396_90597746b8.jpg" alt="Screen shot 2011-01-30 at 22.52.33" width="500" height="281"></a></p>

<p>We have released <a href="https://github.com/komagata/lokka/zipball/v0.2.0">Lokka 0.2.0</a>!</p><p><br>A bunch of cool updates is included in this release;<br></p><ul><li>a default theme “järvi” has been added,</li><li>Theme API and Plugin API are ready,</li><li>Slim template engine is now supported,</li><li>and Lokka.app bundle for Mac users has become available!<br></li></ul><p>Download <a href="https://github.com/komagata/lokka/zipball/v0.2.0">UNIX version</a> or <a href="https://github.com/downloads/komagata/lokka/Lokka-mac-v0.2.0.zip">Mac version</a>.</p>
<hr>

<p>Lokka 0.2.0をリリースしました。</p><p>今バージョンからMac向けにはインストール不要の<a href="https://github.com/downloads/komagata/lokka/Lokka-mac-v0.2.0.zip">Lokka.app</a>が配布されます。</p>

<p>0.1.0からの主な変更点は下記です。</p>

<ul>
<li>標準テーマjärvi(ジャルビ)の追加。</li>
<li>テーマAPIの整備</li>
<li>プラグインAPIの整備</li>
<li>Slimテンプレートへの対応</li>
<li>Mac用Lokka.appのバンドル</li>
</ul>

<p>ダウンロードは<a href="https://github.com/komagata/lokka/zipball/v0.2.0">こちら</a>からどうぞ。(Mac版は<a href="https://github.com/downloads/komagata/lokka/Lokka-mac-v0.2.0.zip">こちら</a>)</p>	Post	2011-01-30 05:30:07
