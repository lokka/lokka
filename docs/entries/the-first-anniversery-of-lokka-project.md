## The first anniversery of Lokka project.

<p>On Arg 10 is The first anniversery of Lokka project.</p>

<p><a href="https://www.flickr.com/photos/komagata/6059435576/" title="Commit History for komagata/lokka - GitHub by komagata, on Flickr"><img src="https://farm6.static.flickr.com/6185/6059435576_01ea1fc9b4_o.png" width="210" height="121" alt="Commit History for komagata/lokka - GitHub"></a></p>

<h3>Why did you made Lokka?</h3>

<p>Because Wordpress doesn’t function in a cloud environment. I really love Heroku, this free, simple cloud (PaaS) service. I made Pyha because I wanted something similar to WordPress that operates in Heroku.</p>

<p>WordPress is useful and I like it a lot. But I used to get annoyed with the lack of consistency in the API. In the summer of 2010 after finishing a long-term project using WordPress, I decided to spend my entire summer vacation redesigning WordPress using Ruby.</p>

<h3>何故Lokkaを作ったのか</h3>

<p>WordPressがクラウド環境で動かないからです。僕は無料で簡単なクラウド(PaaS)サービスのHerokuが大好きです。Herokuで動くWordPress的なモノが欲しかったので作りました。</p>

<p>WordPressは便利で大好きです。しかし一方でAPIの統一感の無さに腹を立てていました。2010年の夏、WordPressを使った長い仕事が終わったとき、Rubyを使ってWordPressをリデザインすることに夏休みを全て費やす決意をしました。</p>
